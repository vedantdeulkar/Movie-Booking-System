# 🎬 Movie Booking System

# 📌 Project Overview
The Movie Booking System is a database-driven SQL project designed to manage movie ticket bookings through an online platform. The system efficiently handles users, movies, theaters, show timings, seat availability, bookings, and payments.
This project showcases strong skills in database design, SQL querying, normalization, and data analysis, with a focus on real-world booking workflows.

# 🎯 Objectives
•	Design an efficient relational database schema for a movie booking platform
•	Manage user information and booking history
•	Store and organize movie, theater, and showtime details
•	Track seat availability and booking status
•	Handle payment records and transaction status
•	Perform CRUD operations across all entities
•	Generate analytical insights using SQL queries

# 📌 Features Implemented
✔ User registration and profile management
✔ Movie database management (genre, language, duration)
✔ Theater and screen management
✔ Show scheduling with date & time
✔ Seat allocation and availability tracking
✔ Ticket booking and cancellation
✔ Booking history and ticket details
✔ Payment tracking (Paid / Pending / Failed)
✔ SQL constraints, relationships, and normalization
